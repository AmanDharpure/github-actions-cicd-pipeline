# Production-Ready CI/CD Pipeline with GitHub Actions

## Project Overview

This project demonstrates a complete CI/CD pipeline for a Node.js Express application using GitHub Actions.

## Features

- Automated dependency installation
- ESLint code quality checks
- Jest and Supertest testing
- Test coverage generation
- Dependency security audit
- Automated application build
- Build artifact upload
- Simulated staging deployment
- GitHub Environment secrets
- Deployment evidence and summary

## Tech Stack

- Node.js
- Express.js
- Jest
- Supertest
- ESLint
- GitHub Actions

## Pipeline Stages

1. Code Quality
2. Automated Tests
3. Dependency Security
4. Build Application
5. Deploy to Staging
